1. in.txt
   LAMMPS input script for model construction.  
2. nep.txt 
   Neuroevolution Potential (NEP) for the Pd-Cu-Ni-P alloy. 
   The content is identical to 'nep.population160.generation1013200.txt'.  
   Reference: https://doi.org/10.1016/j.matdes.2023.112012.  
3. dump.* 
   LAMMPS output structure files with grouped atoms. The lower and 
   upper halves correspond to group 0 and group 1, respectively.  
4. 0.model.*.xyz  
   Extended XYZ (exyz) format files converted from LAMMPS dump files 
   (dump.*) using OVITO.  
5. model.*.xyz 
   Corrected exyz format files generated by modifying '0.model.*.xyz' 
   to ensure proper formatting.  
6. z.sh 
   Job submission script for executing computational tasks.
